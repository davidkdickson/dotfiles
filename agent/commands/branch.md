# Create Branch

## Overview
Create a local git branch using best practices

## Steps
1. **create a local branch using one of the common branch types below
| Type        | When to Use                  | Example                       |
| ----------- | ---------------------------- | ----------------------------- |
| `feat/`     | New features or enhancements | `feat/add-login-flow`      |
| `fix/`      | Regular bug fixes            | `fix/k1-date-parsing`         |
| `chore/`    | Maintenance or cleanup       | `chore/update-ci-config`      |
